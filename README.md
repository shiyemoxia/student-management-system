# 学生学籍管理系统

一个基于Python Flask的学生学籍管理系统，前端采用Vue.js和Bootstrap，后端使用MySQL数据库。系统提供完整的学生学籍信息管理功能，包括学生信息、教师信息、课程管理、成绩管理等核心模块。

## 功能模块

- **学生管理**：添加、编辑、查看和删除学生信息，包括个人基本信息、学籍状态等
- **教师管理**：管理教师基本信息、职称及所属学院
- **课程管理**：课程的增删改查功能，包括课程类型、学分、学时等信息
- **授课管理**：教师授课安排管理，包括学期、教室、上课时间等
- **成绩管理**：学生成绩录入、查看和统计，支持按课程筛选和成绩分析
- **班级管理**：班级和学院的信息管理
- **用户管理**：支持管理员、教师和学生三种角色，权限分级控制

## 系统特点

- **权限控制**：基于角色的访问控制，不同角色拥有不同权限
- **数据安全**：密码加密存储，防止信息泄露
- **响应式设计**：适配不同设备屏幕，提供良好的用户体验
- **数据筛选与排序**：支持多维度的数据筛选和排序功能
- **数据统计分析**：提供学生成绩统计和分析功能

## 技术栈

- **前端**：Vue.js + Bootstrap 5 + Axios
- **后端**：Python Flask + SQLAlchemy
- **数据库**：MySQL
- **认证**：JWT (JSON Web Token)

## 开发环境配置

### 系统要求
- Python 3.8+
- MySQL 5.7+
- Node.js 14+ (仅开发时需要)

### 安装依赖
```bash
pip install -r requirements.txt
```

### 配置数据库
1. 创建MySQL数据库
2. 修改`backend/config.py`中的数据库连接参数
3. 运行`python create_tables.py`初始化数据库和基础数据

### 运行应用
```bash
python run.py
```
应用将在 http://localhost:5000 运行

## 系统账号
- **管理员账号**：admin
- **教师账号**：teacher
- **学生账号**：student
- **默认密码**：123456

## 使用说明

1. **登录系统**：使用提供的账号密码登录
2. **学生管理**：可以添加、编辑、查询学生信息
3. **教师管理**：管理教师信息及职称
4. **课程管理**：管理课程信息，包括课程代码、名称、学分等
5. **授课管理**：安排教师授课，设置上课时间和地点
6. **成绩管理**：录入和查询学生成绩，查看成绩统计
7. **班级管理**：管理班级和学院信息

详细使用说明请参考 [使用说明.md](使用说明.md)

## 数据库设计

系统采用关系型数据库设计，主要包含以下表：
- 学生表(student)
- 教师表(teacher)
- 课程表(course)
- 授课表(offering)
- 成绩表(score)
- 班级表(class)
- 学院表(college)
- 用户表(user)
- 职称表(title)
- 课程类型表(course_type)
